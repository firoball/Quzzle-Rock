Quzzle Rock


This is an Unity Engine learning project with focus on finding a proper project structure
as well as targeting different platforms.
A match three puzzle game was chosen for this.

Android, PC and WebGL deployments were tested.

If you want to make use of the source code or other parts, please respect license.txt
Any included 3rd party content has had its license checked and respected.
